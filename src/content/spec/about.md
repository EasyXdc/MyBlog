# About

Hi, I'm EasyXdc — a developer and graduate researcher who builds backend systems, explores AI/LLM applications, and writes CLI tools for fun.

## What I Do

My work sits at the intersection of **backend engineering**, **AI/LLM**, and **security research**. I build production-grade services with Java and Python, experiment with RAG pipelines and agent frameworks, and occasionally do penetration testing to keep my security instincts sharp.

On the side, I write open-source CLI tools in Rust and Go — the kind of tools that make daily development friction disappear. I believe if you do something more than twice, it deserves a script — or better, a dedicated tool.

By day I work on backend systems, microservices, and DevOps pipelines. By night I tinker with LLM integrations, break things on purpose to figure out how they work, and contribute to open source.
